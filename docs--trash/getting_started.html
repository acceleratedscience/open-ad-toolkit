Preparation

Before You Start

The Toolkit supports the following environments
- Linux
- MacOS
- Windows 11 wsl 2 (ubuntu 22.04)

The Installation will support Python Global, pyenv or Poetry Install. If using Jupyter Notebooks Ensure your version of python is 3.10.10 or greater.
python3 --version
Python 3.11.4

if you wish to Install on Windows 11 WSL you will need to install the Ubuntu WSL package

Install the WSL image and create a user called 'openad' or one of your choosing
wsl --install Ubuntu-22.04

If Running WSL ubuntu or you wish to setup a ubuntu python environment from scratch run the below to setup the environment
- sudo add-apt-repository ppa:deadsnakes/ppa
- sudo apt update
- sudo apt install python3.11-full
- sudo apt install python3-pip
- sudo update-alternatives --install /usr/bin/python3 python3 /usr/bin/python3.11 100
- sudo pip install pip --upgrade


You may also need to setup the default ipython Profile for magic commands you can use ipython profile create

Installing via Poetry

If you have an existing Poetry Project simply run
ppoetry add git+https://github.com/acceleratedscience/open-ad-toolkit.git

Installing via Pip

If you do not want to install in your global environment you can install Openad in a Python Virtual Environment.
To setup and activate a virtual environment for Open Toolkit.
python -m venv ~/openad
source ~/openad/bin/activate

Install Open Toolkit

Install Open Toolkit Framework and its dependencies.
pip install git+https://github.com/acceleratedscience/open-ad-toolkit.git

Starting OpenAD

Start the client

Open a command shell in the client:
openad

Or run a single command:
openad <toolkit_command>

In the command shell, you can print available commands with a single question mark:
?

Use Jupyter Notebooks

If you plan to use it inside a Jupyter notebook or Jupyter Labs:
  1. Create a kernel that makes this environment available inside Jupyter:
    python -m ipykernel install --user --name=ad-kernel
  2. Copy the Toolkit's Jupyter Notebook Examples to your home directory
    init_examples
    This will copy them to ~/openad_notebooks/ directory Once you run the next options run using:
    jupyter lab ~/openad_notebooks/Table_of_Contents.ipynb
  3. Copy the Toolkit's Jupyter-specific magic commands into the environment:
    init_magic
    This will copy the magic commands into the Default jupyter profile so it is automatically available when you start jupyter.
    If for whatever reason there is an error copying the file it is likely that the Default startup directory does not exist.
    you can then run mkdir -p ~/.ipython/profile_default/startup and try init_magic again.
  4. Optional. To copy Jupyter magic commands for use only for a particular notebook, change to the directory with the notebook and run this command:
    init_magic .
  5. Optional, expert. To copy magic commands to an ipython custom profile:
    init_magic <myprofile>

    6. For Those wanting to manually add magic commands to notebooks

  6. Optional, expert. To activate the magic commands in a notebook without running init_magic in the environment:
    Run the command,
    init_examples
    Copy the file ~/openad_notebooks/openad_magic.ipynb to the same directory as your other notebook.
    In your other notebook, run this shell command in a Jupyter Notebook code cell,
    !run openad_magic.ipynb
    This activates the magic commands in that notebook.
