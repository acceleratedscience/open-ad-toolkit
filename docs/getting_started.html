Getting Started

Accelerated Discovery
Open toolkit Framework
v0.01 - IBM Github

Note: if you're on Mac and not installing into a virtual environment you may need use `pip3` and `python3` instead of `pip` and `python` respectively

Preparation

Before You Start:

Ensure you are running python verions 3.10.10 or above.

Step1: Setup a Virtual Enviroment

Run the following to setup a virtual environment to deploy your Open toolkit in.
python -m venv ~/myenv
source ~/myenv/bin/activate
displays command history for the current workspace.

Step2: Install your Open Toolkit Environment

The following will install the Open Toolkit Framework and all it's dependancies.
pip install git+ssh://git@github.ibm.com/Accelerated-Discovery/ad4e-opentoolkit.git

Starting the Client

Now you can deploy the adccl client simply by running adccl or to simply run a you can run adccl <toolkit_command>
Once in the commnd shell you can look at what commands are available by simply typing '?' as a command and you will see all available commands in the shell envrionment.

Working with Jupyter Notebooks

If you plan to use it inside Jupyter Notebook of Jupyter Labs:
-install ipykernel
pip install ipykernel

-Create a kernel that can be used to run notebook commands inside the virtual environment
python -m ipykernel install --user --name=myenv

Then you can Have the Magic Commands Loaded by default by copying the magic commands into the ipython startup directory for your created profile using:
init_magic
Or to copy it for use within only chose notebook do the following to copy it to the current directory and run with !run adccl.py to initiate Magic commands :

init_magic .
Or to copy it to a ipython custom profile

init_magic <myprofile>

Note: if you do not run this command you will need to run it he init_examples and use init_examples command and retrieve the file ~/adccl_notebooks/adccl.ipynb and executer it in each notebook you run with !run adccl.ipynb to activate the magic commands.
