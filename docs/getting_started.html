Preparation

Before You Start

Ensure your version of python is 3.10.10 or greater.
python3 --version
Python 3.11.4

Set up virtual environment

Set up and activate a virtual environment for Open Toolkit.
python -m venv ~/openad
source ~/openad/bin/activate

Install Open Toolkit

Install Open Toolkit Framework and its dependencies.
pip install git+ssh://git@github.ibm.com/Accelerated-Discovery/openad-opentoolkit.git

Start the client

Open a command shell in the client:
openad

Or run a single command:
openad <toolkit_command>

In the command shell, you can print available commands with a single question mark:
?

Use Jupyter Notebooks

If you plan to use it inside a Jupyter notebook or Jupyter Labs:
  1. Install ipykernel
    pip install ipykernel
  2. Create a kernel that makes this environment available inside Jupyter:
    python -m ipykernel install --user --name=openad
  3. Copy the Toolkit's Jupyter Notebook Examples to your home directory
    init_examples
    This will copy them to ~/openad_notebooks/ directory Once you run the next options run using:
    jupyter lab ~/openad_notebooks/Table_of_Contents.ipynb
  4. Copy the Toolkit's Jupyter-specific magic commands into the environment:
    init_magic
    This will copy the magic commands into the Default jupyter profile so it is automatically available when you start jupyter.
  5. Optional. To copy Jupyter magic commands for use only for a particular notebook, change to the directory with the notebook and run this command:
    init_magic .
  6. Optional, expert. To copy magic commands to an ipython custom profile:
    init_magic <myprofile>

    7. For Those wanting to manually add magic commands to notebooks

  7. Optional, expert. To activate the magic commands in a notebook without running init_magic in the environment:
    Run the command,
    init_examples
    Copy the file ~/openad_notebooks/openad.ipynb to the same directory as your other notebook.
    In your other notebook, run this shell command in a Jupyter Notebook code cell,
    !run openad.ipynb
    This activates the magic commands in that notebook.
