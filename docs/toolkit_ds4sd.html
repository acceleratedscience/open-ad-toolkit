Introduction

The Deep Search toolkit integration provides a means to search a Deep Search Repository from the command line or Notebook.
Deep Search leverages state-of-the-art AI methods to continuously collect, convert, enrich, and link large document collections. You can use it for both public and proprietary documents.
To access your deepsearch API key, which is necessary for use with the Opentoolkit see Deep Seearch Website .

Commands:

Display Collections

This function enables you to display available collections to query.
exec display_collection (domain='<domain_name>')



Example Command: exec display_collection( domain='Materials Science' )

Search Repository

This command performs a document search of the Deep Search repository based on a given index. Use Estimate only to determine the potential number of hits.Note when working in a Notebook this command returns a Dataframe.



search for using (index_key=<index_name> [ page_size=<int> elastic_id=<elastic_id> ]) show (data|docs) [ estimate only ] [save as '<csv_filename>'] [return as data]


Note: the 'return as data' clause ensures in notebooks the command only returns a dataframe not a pandas styled object.


The below are examples of how to use the deepsearch "Search" command:

Example 1: this looks for a documents that contain discussions on power conversion efficiency.

search for 'ide("power conversion efficiency" OR PCE) AND organ* ' using (index_key='arxiv' elastic_id=default ) show (docs)

Example 2: searches the pubchem documents for the drug named "Ibuprofen" molecule data

search for 'Ibuprofen' USING (index_key='pubchem') show (data)

Example 3: searches for patents which mention the defined smiles molecule.

search for 'identifiers._name:"smiles#ccc(coc(=o)cs)(c(=o)c(=o)cs)c(=o)c(=o)cs" ' using (index_key='patent-uspto-smiles') show (data)

Example 3: searches for patents which mention the defined smiles molecule and saves it to a csv file

search for 'identifiers._name:"smiles#ccc(coc(=o)cs)(c(=o)c(=o)cs)c(=o)c(=o)cs" ' using (index_key='patent-uspto-smiles') show (data) save as 'myresults.csv'
