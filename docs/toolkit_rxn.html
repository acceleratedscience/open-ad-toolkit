Introduction

The toolkit uses AI to predict outcomes of chemical reactions for optimized synthesis methods, and to automatically generate chemical procedures for use in manual or automated lab operations. The Toolkit is based on molecular transformer models that understand the natural language of chemistry, trained on 2.5 million chemical reactions. These models are flexible, adaptive to new data and non-rule-based.
Before using thr Toolkit log on to https://rxn.res.ibm.com/rxn/sign-in and retrieve your API key from the rxn Web Interface.

Reaction Prediction Commands:

Predict Reaction

This command Forward Predicts a Reaction for a given reaction Smiles string. The Using Clause allows you to set the given ai_model, otherwise it will revert to the default model, The use_saved option will look for a saved result of the command to save re-generating
predict reaction '<reaction-smiles-string>' [USING ( ai_model='<valid_ai_model>')] [use_saved]

Predict Reaction in Batch

This command forward predictions reactions allowing a list from a file or data frame or string list to be processed for predictions in batch displaying the result, all results are saved to the toolkit directory and the [use_saved] clause tells the command to use any already saved reaction predictions .
predict reaction in batch from [dataframe %lt;dataframe_name>] |[file '<file_name.csv>'] | list ['#smiles','#smiles'...] [USING ( ai_model='<valid_ai_model>')] [use_saved]

For example: predict reaction in batch from list ['BrBr.c1ccc2cc3ccccc3cc2c1CCO','BrBr.c1ccc2cc3ccccc3cc2c1']

The valid Options for the USING Clause ai_model=<quoted string> model release, the model defaults to '2020-07-01'

Predict Reaction in Batch topn

This command forward predictions reactions allowing a list from a file or data frame or string list to be processed for predictions in batch displaying the result, all results are saved to the toolkit directory and the [use_saved] clause tells the command to use any already saved reaction predictions .
predict reaction in batch from [dataframe %lt;dataframe_name>] |[file '<file_name.csv>'] | list ['#smiles','#smiles'...] [USING ( topn=<integer> ai_model='<valid_ai_model>')] [use_saved]

For example: predict reaction in batch from list ['BrBr.c1ccc2cc3ccccc3cc2c1CCO','BrBr.c1ccc2cc3ccccc3cc2c1']

The valid Options for the USING Clause
- topn=<integer> top number of reactions.
- ai_model=<quoted string> model release, the model defaults to '2020-07-01'

Predict Retrosynthesis

This command performs automatic retro synthesis on a molecule. predict retrosynthesis '<product_smiles_string>' [USING ( option=<valid_input> option2=<valid_input> )]

Options for USING Clause are:

-availability_pricing_threshold={int} maximum price in USD per g/ml of compounds. Default no threshold.
-available_smiles={quoted string} list of molecules available as precursors. delimiter '.'
-exclude_smiles={quoted string} list of molecules to exclude from the set of precursors. delimiter '.'
-exclude_substructures={str} Substructures to exclude, delimiter '.'
-exclude_target_molecule={bool} excluded Target Molecule, default True
-fap={float} Every retrosynthetic step is evaluated with the FAP, a step is retained when forward confidence is greater than FAP, default 0.6
-max_steps={int} The Maxi steps, default is 3
-nbeams={int} The maximum number of beams exploring the hypertree, default 10
-pruning_steps={int} The number of steps to prune a hypertree, default 2
-ai_model={quoted string} default '2020-07-01'

An example command is:
predict retrosynthesis 'BrCCc1cccc2c(Br)c3ccccc3cc12' " using( max_steps=3 )
